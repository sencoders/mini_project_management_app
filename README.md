# Mini Project Management App

This is a simple full stack project management app built using the MERN stack.  
Users can create projects and manage tasks inside each project.

## Features

- User Signup/Login
- JWT Authentication
- Create/Edit/Delete Projects
- Create/Edit/Delete Tasks
- Change Task Status
- Search Tasks
- Filter Tasks by Status
- Protected Routes
- MongoDB Atlas Integration

## Tech Stack
### Frontend

- React.js
- Vite
- React Router DOM
- Axios
- CSS

### Backend

- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- JWT
- bcrypt.js
